## Note

ESM build only, requires `@midwayjs >= 3.12` and set `"type": "module"` in `packages.json`

## 安装全局依赖
```sh
npm i -g c8 lerna madge rollup tsx zx
```

## Installation
```sh
npm i @waiting/fetch
// for Midway.js
npm i @mwcp/fetch
```
